# Baltimore Budget Project

## Overview
A budget simulator for Baltimore built with Python Dash.

## Tech Stack
- **Framework**: Dash (Plotly)
- **Language**: Python

## Project Structure
```
BaltimoreProject/
├── app.py          # Main Dash application entry point
└── CLAUDE.md       # Project documentation for Claude
```

## Running the Application
```bash
python app.py
```
The app runs on http://localhost:8050 with debug mode enabled.

## Development Notes
- This is a starter project with basic Dash setup
- Budget simulator functionality is planned but not yet implemented

## Commands
- `python app.py` - Start the development server

## Git Workflow
When asked to push changes to GitHub:
1. Stage changes with `git add`
2. Create a clear, descriptive commit message
3. Push to the main branch
4. Confirm the push succeeded

## Deployment
- **Hosting**: PythonAnywhere
- **CI/CD**: GitHub Actions
- Pushing to the main branch on GitHub automatically triggers deployment to PythonAnywhere
